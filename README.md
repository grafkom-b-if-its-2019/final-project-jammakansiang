### Development
Sebelum dibuka di browser, pastikan melakukan build bundle terlebih dahulu:
```bash
# Abaikan jika sudah install
npm install
# Ada penambahan module untuk efek jatuh, wajib update dulu
npm update
npm run build
```
Untuk menjalankan program:
```bash
npm run dev
```
